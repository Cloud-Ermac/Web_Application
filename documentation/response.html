Response

This abstract class represents a Response object, which is used to process and format the data that will be returned to the client in the HTTP response. It has a protected $data property and methods to set and get this data. It also has a protected headers method that can be overridden by child classes to set custom HTTP headers.

Properties

Property Type Description
$data mixed The data that will be returned to the response.

Methods

Method Description
__construct() Calls the headers() method to set any necessary HTTP headers.
headers() Sets the HTTP headers for the response. This method can be overridden by child classes to set custom HTTP headers.
setData($data) Sets the data for the response.
getData() Gets the data for the response.
