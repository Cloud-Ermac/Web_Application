Paper Endpoint

This endpoint allows you to retrieve information about papers at the conference. You can filter the results by short name or by search term.

Endpoint

The endpoint URL is:
unn-w19014555.newnumyspace.co.uk/WebApp/coursework/api/paper

Parameters

The following parameters are accepted:

Name Description
short_name Filters the results by the short name of the paper. Only one short name can be specified at a time.
search Searches for papers that match the specified term. The search is case-insensitive and will match any part of the name of the paper.

Example

The following example retrieves a list of papers with the short name "fullpapers":
/api/paper?short_name=fullpapers

Success Response

If the request is successful, the endpoint will return a JSON object with the following properties:

  • HTTP Status Code: 200 OK
  • JSON body:
{
     {
        "paper_id": "64455",
        "title": "Can Digital Games Teach Scientific Inquiry? A Case of How Affordances Can Become Constraints",
        "award": "false",
        "abstract": "Digital make at least one of four key assumptions about scientific.....
        "short_name": "fullpapers",
        "doi": null,
        "video": "https://www.youtube.com/watch?v=OdYKiNZGgqM"
    },
}

Error Response

If the request is not successful, the endpoint will return a JSON object with the following properties:

  • HTTP Status Code: 400 Bad Request, 404 not found
  • JSON body:
{
      "message": "Invalid combination of short name and search parameters"
}
      {
      "message": "Invalid parameter"
      }
{
    "length": 0,
    "message": "Path not found/paper=short_name=fullpaper",
    "data": null
}
