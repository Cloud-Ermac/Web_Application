Web API Documentation

Author

Description: This endpoint retrieves information about the authors at a conference. It can filter the results based on the `paper_id` or `author_id` parameters.

Request Method: GET

Parameters

  • paper_id (integer, optional): Filters the results by the ID of a paper.
  • author_id (integer, optional): Filters the results by the ID of an author.

Note: Both `paper_id` and `author_id` cannot be provided in the same request.

Example

curl -X GET http://unn-w19014555.newnumyspace.co.uk/WebApp/coursework/api/authors?paper_id=64460

Response

On success:

  • HTTP Status Code: 200 OK
  • JSON body:
{"author_id":"64306","first_name":"Adela","middle_initial":null,"last_name":"Kapuscinska",
    "institution":"Carnegie Mellon University","country":"United States"}

On error:

  • HTTP Status Code: 400 Bad Request
  • JSON body: 
{
    "message": "Invalid parameter [invalid parameter]"
}
or 
{
    "message": "Invalid combination of paper_id and author_id parameters"
}
