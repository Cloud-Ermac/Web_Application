Endpoint: Authenticate

Description: This endpoint allows users to authenticate their username and password against those stored in the database. If authentication is successful, a JSON Web Token (JWT) is returned.

Request Method: POST

Parameters

  • username (string, required): The username to be authenticated.
  • password (string, required): The password to be authenticated.

Example

curl -X POST \
  https://example.com/authenticate \
  -H 'Authorization: Basic [base64-encoded-username-and-password]'

Response

On success:

  • HTTP Status Code: 200 OK
  • JSON body:
{
    "length": 0,
    "message": "success",
    "jwt": "[generated JWT]"
}

On error:

  • HTTP Status Code: 400 Bad Request
  • JSON body:
{
    "message": "Missing required parameters: 'username' and 'password'"
}
